Built with ⚡ **Bun** for performance and supports deployment on **Vercel** or **Cloudflare Workers**.

## 🔌 Running Locally

> **Requirements:**  
> ✅ `Bun` ≥ v1.0.29 **or**  
> ✅ `Node.js` ≥ v20.0.0

### 🔧 Steps:

```bash
# 1. Clone the repo
git clone https://github.com/Shivaay20005/jio-savan-api.git
cd jio-savan-api

# 2. Install dependencies
bun install

# 3. Run the local dev server
bun run dev
